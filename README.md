# Getting Started with Create React App

## react start
npx create-react-app ....   

## axios 설치
npm i axios   

## router 설치
npm i react-router-dom   

## react-player 설치
npm i react-player   

## scss 설치
npm i node-sass   
안 될 경우 : npm i sass   

## react icon 설치
npm install react-icons --save   
아이콘 정보 : https://react-icons.github.io/react-icons
